Hexo搭建

发表于 2016-09-28 | 分类于 technology

系统环境

  • 操作的电脑为macOS系统
  • github账号1枚
  • 域名1枚(此项非必需)

实施步骤

  1. Github相关
  • 申请GitHub账号,此步较为简单，不细说；
  • 设置ssh密钥:



  • 图中Key值从操作系统中获取，方法如下：
    打开终端，输入如下代码
    ssh-keygen -t rsa
    一路回车，然后在当前用户的文件夹下可以发现.ssh文件夹
    由于.ssh文件夹是隐藏文件，首先需要打开macOS的隐藏文件显示，当然也可以直接vi ~/.ssh/id_rsa.pub来查看
    将.ssh文件夹内的id_rsa.pub文件内的内容全部复制出来，添加到

    此图中的Key中
  • 创建以xxx.github.io为名的项目，xxx为你的用户名
  1. 安装Hexo

  • 安装Homebrew
    打开终端，输入以下命令

  • 安装node.js
    安装完Homebrew的前提下
    在终端输入以下命令
    brew install node

  • 安装git
    在终端输入以下命令
    brew install git

  • 安装Hexo
    在终端输入以下命令

    npm install -g hexo
    hexo init
    npm install

  • 设置git

    git config –global user.name “yourname”
    git config –global user.email “youremail@email.com“

  • 修改hexo配置
    打开hexo根目录下的_config.yml文件，修改底部内容为

    deploy:
    type: github
    repo: git@github.com:xxxx/xxxx.github.io.git
    branch: master

  • hexo与github搭配的最最基础使用
    当完成以上配置后，可以使用hexo s来运行，之后访问localhost:4000即可访问自己的Blog了
    当然，此时只是本地的访问，在终端使用

    hexo g
    hexo d

此时访问xxx.github.io，即可看到你的Blog了

  • 域名结合hexo
    此处略去域名的购买
    在域名服务商的网站中，给你的域名添加一条解析，选择CNAME模式，填入blog，目标xxx.github.io
    在hexo的目录中source文件下添加一个名为CNAME的文件(注意名字为大写，无后缀名)，内容为blog.xxxx.xxx,xxxx.xxx为你购买的域名
    完成后在终端输入

    hexo g
    hexo d

之后就能通过blog.xxxx.xxx来访问你的Blog了
